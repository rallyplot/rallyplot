# `rallyplot`

`rallyplot` is an open-source GPU-accelerated plotting Python and C++ library for finance plots.

Leveraging the GPU means plotting millions of candlesticks is now quick and easy. See
below for [installation](installation-page), [getting started](get-started-tutorial) 
and [tutorials](tutorials-index) for a full list of `rallyplot` features.
